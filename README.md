## <a id="anchor5">実行方法</a>
<br>
mac版のminilibxをしようしている関係上動作はMacでしか確認していません。Linux版のminilibxを適用する場合は以下のサイトを参考にしてください。
<br>

>https://github.com/42Paris/minilibx-linux
<br>

実行方法
```
make
./fractol 1 //ジュリア集合
./fractol 2 //マンデルブロ集合
```
